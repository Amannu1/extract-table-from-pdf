# Extract table from pdf

![Java](https://img.shields.io/badge/java-%23ED8B00.svg?style=for-the-badge&logo=openjdk&logoColor=white)

A project built using **Java and Spire** that extracts tables from a specified PDF.

After extracting, the files are compressed into a single ZIP file for easy storage and sharing.
